## What I'm building

**[modelgate](https://github.com/PavanPapiReddy22/modelgate)** — Type-safe, zero-SDK adapter layer that unifies 7 LLM providers (OpenAI, Anthropic, Bedrock, Gemini, Groq, Ollama, Vertex AI) into one consistent async Python interface. Raw `httpx` calls, Pydantic v2 normalization, no vendor lock-in. Published on PyPI. Built as the provider layer powering cinder.

**[cinder](https://github.com/PavanPapiReddy22/cinder)** *(in development)* — Terminal-native AI coding agent with cost-first model routing, built on top of modelgate. Achieves ~80% token reduction vs Claude Code through dynamic tool loading, TTL-capped history, and proactive context compaction at 60% threshold.

```text
Example: "Refactor this module and add tests"

[Orchestrator] ── scores task complexity
    │
    ├── simple tasks  ──► Low-cost model   (file edits, lookups, formatting)
    └── complex tasks ──► High-power model (architecture, reasoning, refactors)
```
